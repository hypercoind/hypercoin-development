## About

btcdiagrams.com is an educational platform featuring Bitcoin infographics and diagrams to help people understand Bitcoin concepts visually. Clear explanations without jargon.

## Features

- Interactive diagram gallery organized by category
- Bitcoin basics, wallets, technical concepts, and economics
- Dark/light theme support
- Responsive design for all devices

## Tech Stack

Built with Next.js and TypeScript, using Astro framework for static site generation for fast loading. Docker deployment ready.
